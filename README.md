# 个人作品集与博客网站

这是一个使用 Next.js、React 和 Tailwind CSS 构建的个人作品集和博客网站。项目采用了独特的像素艺术风格，用于展示个人项目、分享技术文章和提供服务信息。

## 特性

*   **英雄部分**: 突出展示个人或项目亮点。
*   **关于部分**: 介绍个人背景、技能和经历。
*   **服务部分**: 列出提供的服务或专业领域。
*   **作品展示**: 展示精选和全部个人项目，包含详细页面。
*   **博客文章**: 发布最新和全部技术文章或个人思考，包含详细页面。
*   **响应式设计**: 网站在不同设备上均能良好显示。
*   **像素艺术风格**: 独特的视觉风格。
*   **主题切换**: 支持明亮和黑暗模式。

## 技术栈

*   Next.js
*   React
*   Tailwind CSS
*   TypeScript
*   Shadcn/ui (用于 UI 组件)
*   Lucide React (用于图标)
*   Framer Motion (用于动画)

## 安装

请确保您已经安装了 Node.js 和 npm 或 yarn。

1.  克隆仓库:
    ```bash
    git clone <仓库地址>
    ```
2.  进入项目目录:
    ```bash
    cd pixel-portfolio
    ```
3.  安装依赖:
    ```bash
    npm install
    # 或者
    yarn install
    ```

## 运行

运行开发服务器:

```bash
npm run dev
# 或者
yarn dev
```

项目将在 `http://localhost:3000` 上运行。

构建生产版本:

```bash
npm run build
# 或者
yarn build
```

启动生产服务器:

```bash
npm start
# 或者
yarn start
```
